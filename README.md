# 🌍 Projeto Volta ao Mundo — Alemanha 🇩🇪

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de apresentar aspectos importantes da Alemanha de forma visual, organizada e interativa. A proposta faz parte de um trabalho acadêmico, utilizando tecnologias web para criar uma experiência simples e informativa.

O site explora diferentes áreas do país, permitindo ao usuário conhecer mais sobre sua culinária, turismo e esportes.

---

## 🧭 Seções do Site

O projeto está dividido nas seguintes páginas:

* 🏠 **Home**
  Página inicial com uma introdução ao tema e apresentação visual do site.

* 🗺️ **Estados**
  Lista dos estados alemães com suas respectivas capitais.

* 🍽️ **Culinária**
  Apresentação de pratos típicos alemães, com ingredientes e modo de preparo.

* 🏰 **Pontos Turísticos**
  Destaque para locais famosos.


* ⚽ **Esportes**
  Aborda esportes populares na Alemanha.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 4

---

## 🎯 Objetivo

O objetivo principal foi aplicar conhecimentos de desenvolvimento web, especialmente o uso do framework Bootstrap para criação de layouts responsivos e organizados.

---

## 💡 Considerações Finais

Este projeto busca unir design e informação, proporcionando uma navegação agradável e conteúdo relevante sobre a Alemanha.
